# Professional Recruiter-Focused Portfolio Website

A clean, responsive, and minimalist developer portfolio built for **Sahil Raj (Senior Full Stack Developer)**. Designed with a dark-mode-first aesthetic inspired by Vercel, Linear, and GitHub, this site is engineered to load near-instantly, present clean typography, and spotlight impact metrics that appeal directly to recruiters.

**Live Link:** [https://sahilraj3019.github.io/](https://sahilraj3019.github.io/)

---

## 🛠️ Built With
- **Markup**: Semantic HTML5 (structured for accessibility and screen readers)
- **Styling**: Vanilla CSS3 (custom CSS variables, mobile-first responsive layout, dark/light theme toggle)
- **Interactivity**: Pure JavaScript (ES6+) (active scroll highlighting, category tag filtering, lightweight elements)
- **SEO & Discoverability**: Semantic headers, Open Graph meta tags, XML sitemaps, robots.txt crawl rules

---

## ✨ Core Design Features

### 1. Recruiter Impact Metrics
Displays key engineering milestones prominently to capture viewer interest within the first few seconds (4.5+ Years Experience, 100K+ Users Supported, 40% UI Performance Gains, 90% Test Coverage reached).

### 2. Category Project Filters
Allows recruiters to instantly filter projects by tag (e.g., "AI / LLM", "Frontend", "Full-Stack") with fluid, hardware-accelerated transitions.

### 3. Print-Friendly Media Overrides
Includes custom `@media print` style overrides that hide navigation headers, footers, buttons, and decorative icons. This formats the webpage directly into a clean, printable single-page resume layout when printing from a browser (`Ctrl + P`).

### 4. Direct Communication Grid
Replaces bulk input forms with high-contrast link cards for immediate direct routing via LinkedIn, GitHub, CodeSandbox, CodePen, Email, and Phone.
