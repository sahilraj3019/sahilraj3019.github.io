### 📂 Featured Open-Source Projects

#### 🎙️ [Meeting Memo Studio](https://github.com/sahilraj3019/meeting-memo-studio)
- **Tech Stack**: Next.js 14, TypeScript, Tailwind CSS, OpenAI API, Groq.
- **Problem Solved**: Long transcripts from multi-hour video calls are tedious to parse manually, and LLM completions often fail JSON parser validation when schemas drift.
- **Key Features**: Strict JSON-mode LLM integrations paired with a TypeScript schema normalizer for reliable database insertions.
- **Impact**: Provides instant meeting synthesis logs with guaranteed API outputs.

#### 🎽 [Fashion Shot Generator](https://github.com/sahilraj3019/fashion-shot-generator)
- **Tech Stack**: React.js, TypeScript, TensorFlow.js, PoseNet.
- **Problem Solved**: Garment-focused catalog segmentation historically requires uploading full image folders to backend GPU nodes, resulting in high computing costs.
- **Key Features**: pose landmark mapping running purely client-side on standard HTML5 canvas overlays.
- **Impact**: Cuts GPU compute bills to $0 and eliminates upload network overhead.
